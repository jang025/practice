# Array Types

Arrays can be typed using a type annotation followed by empty array brackets, like number[] for an array of numbers

```typescript
let ages: number[] = [1, 2, 3, 4, 5];
```

Alternative Syntax

```typescript
let ages: Array<number> = [1, 2, 3];
```

# Nested arrays

```typescript
const board: string[][] = [
  ["x", "o", "x"],
  ["x", "o", "x"],
];
```
